# Python Practice Questions (For Beginners)

## 1. Variables
Create a variable called `name` and store your name in it. Print it.

## 2. Operators
Make two variables `a = 15` and `b = 5`. Print:
- Their sum
- Their difference
- Their multiplication
- Their division

## 3. Comparison Operators
Create two numbers and check:
- Is the first number greater than the second?
- Is the second number equal to the first?

## 4. Conditions (if-else)
Write a program that checks if a number is greater than 10:
- If yes, print `"Big number"`
- Otherwise, print `"Small number"`

## 5. Even or Odd
Take a number and check:
- If it is even → print `"Even"`
- Otherwise → print `"Odd"`

## 6. For Loop (Counting)
Use a `for` loop to print numbers from 1 to 10.

## 7. Loop Practice
Use a loop to print `"I love Python"` 5 times.

## 8. Lists
Create a list of 3 favorite foods and print each item using a loop.

## 9. Lists + Conditions
Create a list of numbers and print only numbers greater than 10.

## 10. String Formatting
Create variables `name` and `age` and print:
"My name is ___ and I am ___ years old"
using string formatting.
