# 🐍 Python Practice Questions (Loops, Conditions, Lists)

## 🟢 Beginner Level

### 1. Print All Items

Given a list of colors, print each color one by one.

```python
colors = ["red", "blue", "green", "yellow"]
```

---

### 2. Find Even Numbers

Given a list of numbers, print only the even numbers.

```python
numbers = [1, 2, 3, 4, 5, 6]
```

---

### 3. Count Numbers Greater Than 5

Count how many numbers in the list are greater than 5.

```python
numbers = [3, 7, 2, 9, 5, 8]
```

---

### 4. Check if Item Exists

Check if "apple" is present in the list. Print "Found" or "Not Found".

```python
fruits = ["banana", "mango", "apple", "grapes"]
```

---

## 🟡 Medium Level

### 5. Sum of List

Find the total sum of all numbers in the list.

```python

numbers = [10, 20, 30, 40]

```

---

### 6. Find the Largest Number

Find the largest number in the list without using max() .

```python
numbers = [4, 9, 1, 7, 3]
```

---

### 7. Create Squares List

Create a new list with squares of each number.

```python
numbers = [2, 3, 4]
```

---

### 8. Numbers Divisible by 3

Print numbers from the list that are divisible by 3.

```python

numbers = [3, 5, 9, 10, 12]

```

## 🔴 Challenging Level

```python
numbers = [1, 2, 3, 4]
```

Create a new list that contains only positive numbers.

